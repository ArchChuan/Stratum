Stratum Demo 部署架构图

当前 demo 目标是通过 http://101.200.181.141/ 直接访问前端页面； 浏览器请求进入 Traefik Ingress，转发到前端 Nginx，前端再通过 /api/ 反向代理到 Go 后端。

当前访问方式

Traefik 使用 HTTP web 入口，Ingress 不限制 Host，因此浏览器可以直接访问服务器公网 IP。

前后端串联

React 页面由前端 Nginx 提供；所有 /api/ 请求由 Nginx 转发到 stratum:80 后端 Service。

后续域名和 HTTPS

接入域名时恢复 Ingress host、TLS secret、cert-manager 注解，并把 frontendUrl 和 callbackUrl 改成 HTTPS 域名。
